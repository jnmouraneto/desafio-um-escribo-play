2 - A principal função de cálculo, em Dart e em Javascript para testes direto no navegador. 

Utilizei o padrão Model View Controller para separação de tarefas. 
A única view contém um texto explicativo, uma caixa de texto, um botão e um AlertDialog. 

Após o toque no botão, o controlador é acionado para fazer os cálculos e devolver a String que será exibida na AlertDialog.
